YasminOS

lightweight OS for cortex-M0

View project onGitHub

YasminOS is a lightweight Operative System for Cortex-M0+

  1. Compact: less than 2k for kernel
  2. Easy: no gigabyte of source code. Only one C module
  3. No priority: plain round robin scheduling

for usage and building instructions see readme page on github repository
